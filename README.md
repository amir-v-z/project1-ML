<h2 align="right">کار هایی که روی دیتاست marketing انجام شده</h2>

* __فراخوانی کتابخانه ها (import Libraries)__
* __متصل کردن دیتاست__
* __مشاهده شناسنامه داده ها__
    - مشاهده چند سطر اول
    - توضیحات ستون ها
    - تعداد سطر ها و ستون ها
    - نام ستون ها
    - بررسی min و max برای ستون های عددی
    - بررسی مقادیر یکتا در ستون های غیر عددی
    - تعداد Missing Value برای هر ستون
    - توزیع مقادیر (فراوانی هر دسته)
    - آمارتوصیفی پایه برای ستون های عددی
* __تجسم اولیه داده ها (Intial Data Visualization)__
    - نمایش گرافیکی از توزیع و پراکندگی ستون های عددی (نمودار های Histogram , Boxplot)
    - نمایش گرافیکی نمودار میله ای
* __تحلیل همبستگی (Correlation Analysis)__
    - ماتریس همبستگی پیرسون (Pearson Correlation Matrix)
* __داده های پاک (Clean Data)__
    - حذف ستون های غیرضروری
    - تمیز کردن ستون AdSpend
    - بررسی ردیف های تکراری
    - شمارش تکرار در ستون CustomerID
    - نمایش نمودار پراکندگی برای بررسی رابطه بین دو متغیر EmailOpens & EmailClicks
    - تمیز کردن ستون CampaignChannel
* __مدیریت نقاط پرت (Outlier Data Management)__
    - شناسایی و مدیریت نقاط پرت در ستون EmailClicks
    - شناسایی و مدیریت نقاط پرت در ستون Income
* __مهندسی ویژگی (Feature Engineering)__
    - تبدیل ستون Age به گروه‌های رده سنی معنادار
* __پر کردن مقادیر گمشده (Filling in Missing Values)__
    - پر کردن مقادیر گمشده در ستون Income
        - رسم نمودار توزیع درآمد بر اساس سن
    - تصمیم گیری برای حذف یا پر کردن مقادیر گمشده در ستون TimeOnSite
